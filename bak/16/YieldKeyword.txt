Есть ключевое слово yield (переводится как вызов в данном случае).
Если поместить его в тело метода, то потом можно передать любой код при помощи него следующим образом: 

название_метода { код, который вызывается ключевым словом yield }

Так же в сам yield можно передать параметр и поймать затем при помощи пайплайна:

название метода { |параметр преданный yield| код который передается для yield}
