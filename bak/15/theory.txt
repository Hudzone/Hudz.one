Повторяем материал по классам.
Атрибуты.

attr_accessor :var - доступ к переменной как в режиме чтения, так и в режиме редактирования.

attr_reader :var - доступ к переменной только в режиме чтения, менять только через метод.

attr_writer :var - доступ к переменной только в режиме записи.

Переменные внутри атрибута называются свойствами.

Чтобы добавить в массив несколько значений используем знак +=.
arr += [x,y,z]


Наследование (Inheritance).

Class Dog < Animal
End

В этом случае, в классе Dog будут все методы, которые есть в отцовском классе.
Чтобы наследовать несколько классов просто пишем через запятую. 

Чтобы передать атрибуты и инициализацию из родительского класса, в дочернем используем специальный синтаксис:
class Secondary < Parentsl
	def initialize
		super 'variable'
	end
end
