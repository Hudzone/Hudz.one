Чтобы открыть файл:

Var = File.open("название.расширение","режим Открытия")
Режимы:
r - read
w - write
r+ - read + write
w+ - read + write
A - Открыть файл для записи и Передвинуть курсор на конец файла

Метод .write - Используем для открытия файла.
Метод .close - Используем для закрытия файла.
Метод .gets - Используем для Условия и чтения файл строка за строком(читает).

Так же есть возможность переименовать файл:
File.rename 'file1','file2'

В Линукс есть команда chmode параметр файл.расширение
В параметре прописываются доступы к файлу:
R - read
W - write
X - execute

Если вместо параметра цифры от 0 до 7, то эти цифры означает атрибут доступа к файлам:
R W X атрибут
0 0 1 1
0 1 0 2
0 1 1 3
1 0 0 4
1 0 1 5
1 1 0 6
1 1 1 7


Допустим цифра 777, Первая цифра - для владельца файла, вторая цифра - для группы, третья цифра - для пользователя.

В руби как раз есть команда File.chmod три_цифры_доступа.


puts Dir.pwd - напечатать текущую директорию
Dir.chdir 'C:/folder/' - изменить текущую директорию
Dir.entries 'path' - массив всех каталогов
